# Portal MySPD — Next.js

## Mula
```bash
cp .env.example .env.local
npm install
npm run dev
```
Dashboard dan login telah dimigrasi ke Next.js App Router. Modul lama dikekalkan di `public/legacy` dan dipetakan melalui rewrites agar fungsi sedia ada tidak terputus. Pindahkan modul tersebut ke React satu demi satu. Pembaikan Borang 4 Jam turut mengekalkan keputusan terkini supaya cetakan masih tersedia selepas tarikh bertukar.
